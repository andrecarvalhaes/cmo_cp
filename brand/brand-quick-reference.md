## 1. Regras Visuais (obrigatórias)

| Elemento | Regra |
|----------|-------|
| **Cor primária** | `#F26600` (laranja) — DEVE aparecer em toda peça |

| **Fundo** | Branco `#FFFFFF` ou cinza claro `#F6F7F9` |

## 2. Tom de Voz

### Somos
- **Simpático** — acessível, nunca arrogante
- **Relevante** — conteúdo que importa
- **Direto** — sem rodeios, objetivo
- **Esclarecedor** — explica com clareza

### Regras de escrita
- Sempre **"o ClubPetro"** (artigo masculino)
- Tratar como **"você"** (singular, intimista)
- Tom **coloquial mas correto** — sem gírias
- Foco em **solução**, não no problema
- Usar **"nós"/"a gente"** para incluir o público
- Evitar clichês — ser criativo
- Demonstrar **domínio** do tema

### NÃO fazer
- NÃO ser seco, distante ou prolixo
- NÃO usar gírias ou exagerar informalidade
- NÃO subestimar a inteligência do leitor
- NÃO prometer o que não podemos cumprir

### Tom por canal
| Canal | Tom |
|-------|-----|
| **Blog** | Reflexivo, opinativo, perguntas provocativas, dados. "Você" (80%) |
| **Email** | Newsletter = informativo. Aulão = criativo + urgência. Nurturing = educacional |
| **Social** | Leve, direto, visual, interação |
| **Interno** | Estratégico, alinhamento. Petrinho e Speeders permitidos |

## 3. Glossário Rápido

### USAR
| Termo | Contexto |
|-------|----------|
| **revendedor** | Sempre (84% dos posts). Nunca "dono de posto" |
| **frentista** | Funcionário da pista. Nunca "atendente de pista" |
| **solução** / **ferramenta** | Referir ao produto. Nunca "software" |
| **"o ClubPetro"** | Masculino. Nunca "a ClubPetro" |
| **Aulão** | Webinar. NUNCA "webinar" ou "live" |
| **Conexão Revenda** | Evento presencial |
| **posto** / **revenda** | Estabelecimento do cliente |

### NÃO USAR (externo)
| Termo | Por quê |
|-------|---------|
| ~~dono de posto~~ | 0% de uso no blog — usar "revendedor" |
| ~~software~~ | Nunca. Usar "solução" ou "ferramenta" |
| ~~plataforma~~ | Evitar (6%). Usar "solução" |
| ~~ecossistema~~ | Compete com Linx no mercado |
| ~~aceleradora~~ | Apenas interno |
| ~~Speeders / Petrinho~~ | APENAS comunicação interna |
| ~~webinar / live~~ | Usar "Aulão" |

### Tratamento
- **"você"** (singular, intimista) — padrão
- **"nós"** (inclusivo) — para incluir o leitor
- **"a gente"** — reservar para social informal, NÃO usar no blog

## 4. Posicionamento

**Essência:** "Aceleradora de resultados para postos"

| Contexto | Linguagem |
|----------|-----------|
| **Interno** | "Aceleradora de resultados" |
| **Site** | "Aditivamos os resultados do seu posto" + métrica |
| **Revendedor** | "Parceiro de resultados" / "Solução completa" |
| **Evitar externo** | "Ecossistema" |

**Tagline:** "Poder para o revendedor"

**Diferencial:** Única empresa do setor que quantifica a promessa ("20% do volume em até 90 dias").

## 5. Fotografia

- **Pessoas**: sorrindo ou confiantes, humor alinhado ao tema
- **Ambiente**: bombas, frentistas, carros — usar o mundo do posto
- **Dispositivos**: templates com telas do produto
- Comunicação imagética: **clara e objetiva**
